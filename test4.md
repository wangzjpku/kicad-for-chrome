# Test4.md - 手动测试完整流程

本文档描述通过浏览器界面手动测试 kicad-for-chrome 项目的完整流程。

## 测试环境准备

### 1. 启动后端服务

```bash
# 进入后端目录
cd kicad-ai-auto/agent

# 激活虚拟环境 (Windows)
venv\Scripts\python main.py

# 或 (Linux/Mac)
source venv/bin/activate
python main.py
```

**验证后端启动成功:**
- 访问 http://localhost:8000/docs
- 确认 API 文档页面加载正常
- 检查终端无报错

### 2. 启动前端服务

```bash
# 进入前端目录
cd kicad-ai-auto/web

# 安装依赖 (如首次运行)
npm install

# 启动开发服务器
npm run dev
```

**验证前端启动成功:**
- 访问 http://localhost:3000
- 确认主页加载正常
- 检查浏览器控制台无报错
